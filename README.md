# Whistle2

A macOS menu bar app that detects whistle sounds.

## Features

- Real-time whistle detection
- Menu bar interface for easy access
- SwiftUI-based user interface

## Requirements

- macOS 13.0 or later
- Xcode 14.0 or later (for building from source)

## Installation

1. Clone this repository
2. Open the project in Xcode
3. Build and run the app

## Usage

The app runs in your menu bar and provides whistle detection functionality. Click the whistle icon in your menu bar to access the interface.

## Development

This project is built with:
- SwiftUI for the user interface
- Swift for the core functionality
- macOS frameworks for menu bar integration
